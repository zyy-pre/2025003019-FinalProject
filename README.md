# 今日灵感 - 每日壁纸与名言收藏器

一个简洁的 Android 应用，每天从网络获取一张美图和一句名言，喜欢就收藏到本地。

## 功能

- 首页展示随机壁纸和名言
- 一键收藏喜欢的内容
- 支持搜索收藏
- 查看收藏详情，编辑标签
- 深色/浅色模式支持

## 技术栈

- Kotlin + Jetpack Compose
- Material 3
- Room 数据库
- DataStore
- Retrofit + OkHttp
- Coil 图片加载
- Navigation Compose
- Kotlin Coroutines

## API 来源

- 名言：https://api.quotable.io/quotes/random
- 图片：https://picsum.photos

## 运行

1. 使用 Android Studio 打开项目
2. 等待 Gradle 同步
3. 运行到模拟器或真机
